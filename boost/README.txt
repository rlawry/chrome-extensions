YouTube Volume Boost Chrome Extension

Files:
- manifest.json
- content.js
- popup.html
- popup.js

Install locally in Chrome:
1. Open chrome://extensions
2. Turn on Developer mode
3. Click Load unpacked
4. Select this folder
5. Open YouTube and use either the popup slider or the floating on-page GUI to set the boost

Notes:
- Range is 100% to 500%
- High gain can cause clipping or distortion
- Works on youtube.com and music.youtube.com
- The floating GUI can be hidden and re-opened from its compact badge
